# Sorting Algorithms Visualizer 
# Implemented algorithms: 
1. Selection Sort
2. Bubble Sort
3. Insertion Sort
4. Merge Sort
5. Quick Sort(Using Hoare's Partition)
6. Heap Sort
7. Shell Sort
8. Cycle Sort
9. Counting Sort
10. Radix Sort
11. Bucket Sort

# To Run on your OS:
1. First FORK this repository on your github and then CLONE it using git clone command
2. run npm install
3. run npm start
